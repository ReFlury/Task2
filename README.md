# Ansible: Настройка Web-сервера и Firewall

Этот репозиторий содержит Ansible-плейбук для автоматической настройки веб-сервера Nginx и защиты его с помощью UFW.

## Структура репозитория
site.yml — основной плейбук
inventory.ini — список серверов
templates — папка с шаблонами Jinja2:
    index.html.j2 — шаблон главной страницы
    nginx.conf.j2 — конфигурационный файл Nginx

## Как запустить

1. Подготовка:
   Убедитесь, что у вас установлен Ansible и есть доступ по SSH к целевым серверам.

2. Настройка:
   Отредактируйте файл inventory.ini, указав IP-адрес вашего сервера:
   [web]
   192.168.1.X ansible_user=ваш_юзер

3. Запуск: Выполните команду в корне репозитория:
   ansible-playbook -i inventory.ini site.yml -K
(Флаг -K запросит пароль sudo для повышения привилегий)

## Ожидаемый результат
После успешного выполнения:
1. На сервере будет запущен Nginx
2. По адресу http://IP-СЕРВЕРА откроется страница с информацией о системе
3. Firewall (UFW) будет активен:
SSH (22): открыт для всех
HTTP (80): открыт только для указанной подсети
